Figma Naming Guide

ForgeGUI · keep this open beside Figma

ForgeGUI builds the course shell from your Figma frame by reading layer names. Name layers with the prefixes below and buttons + dynamic-text zones wire up automatically. Matching is forgiving — case, spaces, and - vs _ don't matter, and a name only has to contain the keyword.

1 · Stage frame

Name the frame to import stage, gui, or shell.

3 · Text zones — zone-<keyword>

The importer picks the closest type from a keyword in the name:

Name contains…TypeShows
lessonLesson titlelesson name
frame / titleFrame titleframe name
section / moduleSection titlecourse/section name
countFrame countere.g. 3 / 17

feedbackFeedbackquiz correct/incorrect
(other)Promptfallback
Standard title stack
top line → zone-lesson-name → the Lesson (e.g. "Welcome")
second line → zone-frame-name → the Frame (e.g. "Welcome to CourseForge")

Recommended exact names: zone-lesson-name · zone-frame-name · zone-section-name · zone-frame-counter · zone-prompt · zone-feedback

  • Use a Figma text layer for zones — its size, alignment, and color carry over.
  • Give each zone its own keyword; don't reuse a purpose.
  • You can override any zone's Type in the ForgeGUI editor — the selected zone shows a marching-ants outline on the stage so you know which one you're editing.
